# Group Policy Configuration

## Policy Used

The environment uses the Default Domain Policy to enforce domain-wide account security settings.

---

## Configured Settings

The following settings were configured.

### Password Policy

- Minimum password length
- Password complexity requirements
- Password expiration policy

---

### Account Lockout Policy

- Account lockout threshold
- Lockout duration
- Reset lockout counter time

---

## Purpose

These policies enforce baseline domain security controls by ensuring that user accounts follow strong authentication requirements.

Group Policy allows centralized configuration of security settings across all domain users and computers.
