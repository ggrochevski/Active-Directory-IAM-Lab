# Security Logging

## Logging Scope
Security logging was partially implemented to support identity and activity visibility.

## Configured Logging
- Advanced Audit Policy
- Logon events
- Process creation events
- Event ID `4688`

## PowerShell Logging
PowerShell Script Block Logging and Event ID `4104` were part of the broader lab and detection discussions connected to the environment.

## Purpose
These logging settings improve visibility into:
- authentication activity
- account-related events
- process execution
- PowerShell activity
